<p>本项目用到了selenium和 GeneralNewsExtractor抽取部分垂直行业新闻并进行展示。其中：</p>
<ol>
<li>栏目资源列表的定位主要基于各具体网站的HTML结构，需要根据实际情况（网站改版）进行调整；</li>
<li>主要根据特征字串以及后期生成的分类来过滤栏目资源列表；</li>
<li>新闻初步分类的生成使用了百度智能云的NPL能力；</li>
<li>新闻摘要生成使用了火山方舟大模型服务能力；</li>
<li>综合分析抽取网站和栏目的性质、标题、摘要内容、初步分类和子分类等内容，来生成生成最终分类，并可进行手工校核；</li>
<li>抽取的信息存放在sqlite本地库中，可以根据需要换成其他数据库；</li>
<li>基于jinja2生成一定时间段的静态网页（如一周），并根据最终分类展示。</li>
</ol>
